# Evaluating-NWM-3.0-Peak-Streamflow-Against-USGS-Gauges
Python workflow for evaluating National Water Model 3.0 peak streamflow performance against USGS gauges in Oklahoma. Uses event-based peak matching, site‐level skill metrics (KGE, NSE, PBIAS, RMSE), and spatial / sensitivity analyses to compare regulated vs unregulated basins.

This repository contains a reproducible Python workflow for evaluating National Water Model (NWM) 3.0 simulated instantaneous peak streamflow against USGS observed peaks across the Oklahoma streamgage network. The notebook extracts annual peak events, performs event-based peak matching within configurable time windows, and computes site-level performance metrics including KGE, NSE, PBIAS, and RMSE. Results are summarized by regulated vs unregulated basins and drainage-area classes, with outputs that include site tables, spatial maps, and sensitivity analyses across threshold and window settings. The code is organized so users can easily point to new input directories, rerun the analysis, and regenerate all figures and CSV outputs.
