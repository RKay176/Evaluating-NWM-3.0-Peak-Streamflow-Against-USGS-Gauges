# Evaluating-NWM-3.0-Peak-Streamflow-Against-USGS-Gauges
Python workflow for evaluating National Water Model 3.0 peak streamflow performance against USGS gauges in Oklahoma. Uses event-based peak matching, site‐level skill metrics (KGE, NSE, PBIAS, RMSE), and spatial / sensitivity analyses to compare regulated vs unregulated basins.
